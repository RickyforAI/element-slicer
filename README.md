# 元素切切乐 (Element Slicer)

一个有趣的化学元素教育游戏，通过切割飞行的元素来学习化学知识。

## 🎮 游戏特色

- **教育性强**：在游戏中学习化学元素知识
- **四种元素类型**：普通、毒雾、幻影、爆裂元素
- **连击系统**：连续切中获得额外分数
- **渐进难度**：随着游戏进行难度逐渐提升
- **移动端支持**：完美适配手机和平板

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html`
2. 或访问在线版本：[即将上线]

## 🎯 游戏玩法

- **目标**：在规定时间内达到300分
- **操作**：鼠标滑动或手指滑动切割元素
- **注意**：不同元素有不同效果，小心爆裂元素！

## 🧪 元素类型

| 类型 | 元素 | 颜色 | 效果 | 得分 |
|------|------|------|------|------|
| 普通 | C, N, O, H | 蓝色 | 正常消失 | +5分 |
| 毒雾 | F, Cl, Br | 绿色 | 释放毒雾3秒 | +8分 |
| 幻影 | He, Ne, Ar | 青色 | 需切3次 | +15分 |
| 爆裂 | Li, Na, K | 红色 | 爆炸减生命 | 0分 |

## 🛠️ 技术栈

- **Phaser 3.70.0** - 游戏引擎
- **原生JavaScript** - 无需构建工具
- **HTML5 Canvas** - 渲染技术

## 📱 系统要求

- 现代浏览器（Chrome, Firefox, Safari, Edge）
- 支持触摸的移动设备
- 无需安装，开箱即用
